배열

자바스크립트에서는 자료형 지정이 없기 때문에 한 배열 안에 모든 자료형 보관 가능

배열 선언 및 초기화

배열 선언 시 크기를 지정하지 않고 선언할 수도 있고 크기를 정하고 선언할 수도 있음

배열 메소드

array도 하나의 객체이기 때문에 배열에서 활용할 수 있는 메소드가 있음

indexOf()

배열에서 요소가 위치한 인덱스 리턴

concat()

여러 개의 배열 결합

join()

배열의 내용을 문자열로 반환

reverse()

배열의 순서를 뒤집음

sort()

배열 내림차순 혹은 오름차순으로 정렬

push()와 pop()

push() : 배열의 맨 뒤에 요소 추가
pop() : 배열의 맨 뒤에 요소 제거

shift()와 unshift()

shift() : 배열의 맨 앞에 요소 제서
unshift() : 배열의 맨 앞에 요소 추가

slice와 splice()

slice() : 배열의 요소 뽑아내기
splice() : 배열의 index위치 요소 제거 및 추가

toString()

배열을 문자열로 반환
