시멘틱 태그

컴퓨터는 단순 문장이나 문장 위치, 모양만으로는 내용을 이해하기 힘드므로 컴퓨터가 각 문장에 대한 의미를 이해할 수 있도록 요소를 사용하여 명확하게 나타내기 위함

header

body 안에 사용하며 여러 번 사용 가능
웹 문서에서 머리말 영역을 나타 낼 때 사용(제목, 제목 설명, 문장 등을 나타내는 요소 포함)
menu1 menu2 navigation, 웹 문서 내 다른 웹문서나 문서 내 다른 부분으로 이동하는 부분을 나타내고자 할 때 사용

aside
